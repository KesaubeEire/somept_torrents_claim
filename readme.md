### 使用方式

1. 打开 kame 的个人页面 (就顶部写着自己 id 的部分)
2. 找到 "当前做种" 这一栏, 点开 \[显示/隐藏\] 这个项目, 不打开点按钮是没用的捏
3. 点击红色的 "一键认领", 等待响应, 种子越多响应时间越长的捏
4. 等到有 alert 面板弹出来就结束了捏
